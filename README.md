# LuxAuto

Aplicação web interativa que simula o ambiente digital de uma concessionária de veículos de alto padrão. O site foi projetado para guiar o usuário desde a descoberta do produto até a simulação financeira.

1.Vitrine Interativa e "Lifestyle":

Além de um carrossel tradicional para destaques, o site apresenta uma seção de "Estilo de Vida" onde os veículos são exibidos lado a lado.
Ao interagir (passar o mouse), os veículos reagem visualmente (efeito de zoom e destaque), incentivando a exploração dos modelos disponíveis.

2.Configurador de Veículos (Modais):

O sistema permite que o usuário explore detalhes de cada carro sem sair da página principal, utilizando janelas sobrepostas (Modais).

Dentro do configurador, há uma divisão estratégica em abas:
Aba Configuração: Permite trocar a cor do veículo instantaneamente através da manipulação do DOM.
Aba Técnica: Exibe dados de engenharia (blueprint, dimensões e potência) em um layout técnico.
